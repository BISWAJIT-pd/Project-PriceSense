# 📊 Project PriceSense
### A SQL-Driven Pricing Intelligence Framework

PriceSense is an end-to-end SQL analytics project that helps D2C nutrition brands make data-driven pricing decisions. By analyzing transactional sales data, customer purchasing behavior, product performance, and market trends, the project identifies optimal pricing strategies that maximize revenue while maintaining customer demand.

Instead of relying on intuition, PriceSense transforms raw business data into actionable insights through advanced SQL queries and business intelligence techniques.

---

## 🚀 Project Objective

The objective of this project is to answer critical business questions such as:

- What is the optimal price range for each product?
- Which customer segments are most price-sensitive?
- Which products generate the highest revenue?
- How does pricing vary across customer personas?
- Which regions contribute the highest sales?
- Which products maintain demand despite higher prices?
- How do seasonal trends affect pricing and sales?
